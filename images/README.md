## Image Optimization Guidelines

### Format Standards

#### WebP Format (Primary)
- **Compression**: 80-85% quality
- **Use Case**: Modern browsers with WebP support
- **Benefits**: 25-35% smaller file size than JPEG
- **Naming Convention**: `image-name.webp`

#### JPEG Format (Fallback)
- **Compression**: 85-90% quality
- **Use Case**: Browsers without WebP support
- **Color Space**: sRGB
- **Naming Convention**: `image-name.jpg`

### Responsive Image Sizes

Generate multiple sizes for each image to serve appropriate dimensions based on viewport:

- **Small (Mobile)**: 640px width
- **Medium (Tablet)**: 1024px width
- **Large (Desktop)**: 1600px width
- **Extra Large (HD)**: 1920px width

### File Size Targets

- **Hero Images**: < 200KB (WebP), < 300KB (JPEG)
- **Section Images**: < 150KB (WebP), < 200KB (JPEG)
- **Portrait Images**: < 100KB (WebP), < 150KB (JPEG)
- **Thumbnail Images**: < 50KB (WebP), < 75KB (JPEG)
